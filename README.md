## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Dart SDK installed
- Android Studio or VS Code

### Installation
1. Clone the repository
   git clone https://github.com/zaidhindi/store_app.git
2. Navigate to the project folder
   cd store_app
3. Install dependencies
   flutter pub get
4. Run the app
   flutter run
